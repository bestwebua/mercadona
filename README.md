## Requirements

Ruby MRI 2.5.0+

## Installation

Add this line to your application's `Gemfile`:

```ruby
gem 'mercadona'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install mercadona
```
